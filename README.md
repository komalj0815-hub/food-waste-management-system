
# Local Food Wastage Management System

## Project Overview

The Local Food Wastage Management System is designed to reduce food waste by connecting food providers with receivers through a centralized platform.

## Technologies Used

- Python
- PostgreSQL
- Streamlit
- Pandas
- Plotly

## Features

- Food Listings Management
- Claims & Distribution Analysis
- Provider and Receiver Insights
- Interactive Dashboards
- Data Filtering
- Food Waste Analytics

## Project Structure

- Food Listings & Availability
- Claims & Distribution
- Analysis & Insights
- Dashboard Overview

## Deployment

Developed using Streamlit and PostgreSQL database.
